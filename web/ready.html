Editor de Código

Instalar VS Code

Sitio oficial — code.visualstudio.com
1
Abre el navegador y visita el sitio oficial
Ve a code.visualstudio.com. El sitio detectará automáticamente tu sistema operativo (Windows, macOS o Linux) y te mostrará el botón de descarga correcto.
2
Descarga el instalador
Haz clic en el botón azul "Download for Windows" (o el de tu sistema). Se descargará un archivo .exe (Windows), .dmg (macOS) o .deb/.rpm (Linux).
3
Ejecuta el instalador
Abre el archivo descargado y sigue el asistente. Importante: en Windows marca la opción "Add to PATH" y "Open with Code" para facilitar el uso desde la terminal.
4
Abre VS Code
Una vez instalado, abre VS Code desde el menú de inicio o buscándolo en tu sistema. Verás la pantalla de bienvenida con el fondo de código binario.
ℹ️
¿Por qué VS Code? Es el editor más popular del mundo, gratuito, open-source y con soporte nativo para Python, terminal integrada y depurador. QuantLab fue desarrollado con VS Code.

Lenguaje de Programación

Instalar Python 3.13

Python 3.13 — Microsoft Store / python.org
1
Opción A — Microsoft Store (Windows, más fácil)
Abre la Microsoft Store y busca "Python 3.13". Haz clic en "Obtener". Se instalará automáticamente sin configuración adicional.
2
Opción B — Instalador oficial (todas las plataformas)
Ve a python.org/downloads y descarga Python 3.13. En el instalador de Windows, marca obligatoriamente la casilla "Add Python to PATH" antes de instalar.
3
Opción C — macOS con Homebrew
Si usas Mac, abre la Terminal y ejecuta el siguiente comando:
# macOS — instalar Homebrew si no lo tienes
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Luego instalar Python 3.13
brew install python@3.13
4
Verifica la instalación
Abre una terminal (en VS Code: Terminal → New Terminal o Ctrl+`) y ejecuta:
python --version
# Deberías ver: Python 3.13.x

pip --version
# Deberías ver: pip 24.x.x from ...
⚠️
Si el comando no funciona en Windows: En algunos sistemas el comando es python3 en lugar de python. Prueba con python3 --version. Si sigue fallando, reinstala Python marcando "Add to PATH".

VS Code

Instalar Extensiones

Extensions Marketplace — Python Debugger by Microsoft

Las extensiones de VS Code agregan soporte completo para Python: resaltado de sintaxis, autocompletado inteligente, depurador visual y selección de intérprete.

1
Abre el panel de extensiones
Presiona Ctrl+Shift+X (Windows/Linux) o Cmd+Shift+X (macOS). Se abrirá el marketplace de extensiones en el panel izquierdo.
2
Instala "Python" de Microsoft
Busca "Python" y selecciona la extensión oficial de Microsoft (la primera, con más de 100M de descargas). Haz clic en Install.
3
Instala "Python Debugger" de Microsoft
Busca "Python Debugger" — también de Microsoft. Como se ve en la imagen, esta extensión usa debugpy e incluye el depurador completo. Haz clic en Install.
4
Selecciona el intérprete de Python
Presiona Ctrl+Shift+P, escribe "Python: Select Interpreter" y selecciona Python 3.13 de la lista. Esto conecta VS Code con tu instalación de Python.
✅
Extensiones recomendadas adicionales: También puedes instalar Pylance (autocompletado avanzado) y Python Indent para una mejor experiencia de desarrollo con QuantLab.

Instalar Dependencias

QuantLab usa varias librerías especializadas. Abre la terminal integrada de VS Code (Ctrl+`) y ejecuta los comandos a continuación.

Librería Versión Uso en QuantLab
flask≥ 3.0Servidor web — sirve la interfaz gráfica en el navegador
yfinance≥ 0.2Descarga datos de mercado en tiempo real (NASDAQ, BTC, etc.)
numpy≥ 1.26Cálculos vectoriales — FFT, Kalman, estadísticas
pandas≥ 2.1Manejo de series de tiempo y DataFrames de precios
plotly≥ 5.18Gráficos interactivos — charts de precios, espectro Fourier
scipy≥ 1.11Detección de extremos locales (soportes/resistencias), stats
💡
Instalación en un solo comando: Puedes instalar todas las dependencias de una sola vez con el comando de abajo.
# Instalar todas las dependencias de QuantLab de una vez
pip install flask yfinance numpy pandas plotly scipy

# Si usas python3 en tu sistema:
pip3 install flask yfinance numpy pandas plotly scipy

¿Tardó la instalación? Es normal. NumPy, SciPy y Plotly son librerías grandes. La primera instalación puede tardar entre 1-5 minutos dependiendo de tu conexión a internet.

Una vez que termine verás el mensaje Successfully installed seguido de los nombres de los paquetes.

# Verifica que todas las dependencias están instaladas:
pip list | grep -E "flask|yfinance|numpy|pandas|plotly|scipy"

# Deberías ver algo así:
# Flask          3.x.x
# numpy          1.x.x
# pandas         2.x.x
# plotly         5.x.x
# scipy          1.x.x
# yfinance       0.x.x
⚠️
Error "pip no reconocido": Si pip no está disponible, ejecuta primero python -m ensurepip --upgrade y luego vuelve a intentar la instalación.
🔴
Entorno virtual (recomendado para proyectos grandes): Si planeas tener múltiples proyectos Python, es buena práctica usar un entorno virtual para aislar las dependencias. Ejecuta: python -m venv venv y luego actívalo con venv\Scripts\activate (Windows) o source venv/bin/activate (macOS/Linux).

Ejecutar el Sistema

Correr QuantLab

Con VS Code, Python y todas las dependencias instaladas, ya estás listo para ejecutar QuantLab. Sigue estos pasos finales:

1
Coloca los archivos del proyecto
Crea una carpeta llamada QuantLab en tu computadora (por ejemplo en el Escritorio). Dentro de ella coloca los dos archivos: Anomalias_Web.py e ingeneria.py. Ambos deben estar en la misma carpeta.
2
Abre la carpeta en VS Code
En VS Code ve a File → Open Folder y selecciona la carpeta QuantLab. Verás los dos archivos en el panel izquierdo del explorador.
3
Abre la terminal integrada
Presiona Ctrl+` (acento grave) o ve a Terminal → New Terminal. Se abrirá una terminal apuntando a la carpeta del proyecto.
# Navega a la carpeta del proyecto (si no estás ya en ella)
cd C:/Users/TuUsuario/Desktop/QuantLab   # Windows
cd ~/Desktop/QuantLab                    # macOS / Linux

# Ejecuta el servidor QuantLab
python Anomalias_Web.py

# Si usas python3:
python3 Anomalias_Web.py
4
Confirma que el servidor arrancó
Verás en la terminal la salida del sistema con los mercados y herramientas disponibles, seguido de la línea Running on http://0.0.0.0:5000.

Solución de Problemas

Errores Comunes

!
ModuleNotFoundError: No module named 'flask'
Las dependencias no están instaladas correctamente. Verifica que estás usando el mismo Python donde instalaste los paquetes. Ejecuta pip install flask yfinance numpy pandas plotly scipy de nuevo.
!
Address already in use (puerto 5000 ocupado)
Ya hay un proceso usando el puerto 5000. Cierra otras instancias de QuantLab o cambia el puerto en la última línea de Anomalias_Web.py: port=5001
!
No data fetched / Error obteniendo datos
Problema de conexión a internet o yFinance. Verifica tu conexión, espera unos segundos y vuelve a analizar. Los mercados fuera de horario pueden devolver datos limitados.
!
ImportError en macOS: ingeneria no encontrado
Verifica que ingeneria.py y Anomalias_Web.py están en la misma carpeta y que ejecutas el comando desde esa carpeta en la terminal.
# Comando completo de diagnóstico — ejecuta esto si algo falla
python -c "import flask, yfinance, numpy, pandas, plotly, scipy; print('✓ Todas las dependencias OK')"

# Si ves el mensaje verde, todo está instalado correctamente.
# Si ves un error, reinstala el paquete que falla:
pip install NOMBRE_DEL_PAQUETE --upgrade
